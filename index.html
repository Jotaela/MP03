Servidor d'actualitzacions

  • Aquesta pàgina estaa dedicada el servidor WSUS, un servei de windows Active Directory orientat a gestionar i administrar les còpies de seguretat dels nostres equips windows units al domini:

    1 Instal·lació:

    • Per a instal·lar el servei haurem d'acudir un altre cop a Agregar roles y características. En aquest cas el rol serà Windows Server Update Services:

    • En aquest pas li assignarem la unitat de disc que hem instal·lat en el nostre sistema previament:
    • Quan finalitzi la instal·lació executarem la configuració posterior simplement fent clic sobre Iniciar tareas posteriores a la instalación:

    2. Configuració:

    • Ara ja podrem accedir a Windows Server Update Services.
    • Per a configurar el nostre WSUS ens mourem a Opciones -> Asistente para la configuación del servidor WSUS:
    • Durant la configuració seguirem les següents captures:


    • Aquí triarem els diferents idiomes en els que volem al actualització:
    • Aquesta pantalla definirà quines son les actualitzacions que volem desar:



    • Un cop hagi acabat sol ens quedara esperar a que sincronitzi:
    • Ara passarem un moment al client i configurarem les actualitzacions per a que no les cerque:
    • Tornem al servidor, on aquest cop, dins de l'eina WSUS aprobarem totes les actualitzacions tal com es motra en les següents captures:

    • En aquesta pantalla aprobarem les diferents caselles amb clic-dret -> Apreobada para su instalación:
    • I acceptarem tots els termes de llicencia...
